# Project Configuration

NodeJS, Express.js & postgresql.

## Engines

- Required node >= v20.0.0
- Required npm >= v9.6.4

## Bais Commands

- npm run start (Start the server)
- npm run migrate:fresh (Migrate all tables)
- npm run migrate:drop (Drop all tables)
- npm run master:seed (Seed data)
- npm run master:seed:undo (Remove seeded data)
